TEMPLATE FORTRAN FILES (template_variant/)
===========================================

All files are free-form Fortran 90, fully documented, with placeholder coefficients.

1. template_crown.f90 (285 lines)
   - Crown ratio prediction model
   - Equation: CR = 10*(BCR1/(1+BCR2*BA) + BCR3*(1-exp(BCR4*DBH)))
   - Coefficients: BCR1, BCR2, BCR3, BCR4 (4 per species)
   - Placeholders marked with TODO comments

2. template_dgf.f90 (320 lines)
   - Diameter growth function
   - Equation: ln(DDS) = b0 + b1*ln(DBH) + b2*BA + b3*SI + COR
   - Coefficients: B1, B2, COR (3 per species)
   - Includes annual growth calculation subroutine

3. template_morts.f90 (280 lines)
   - Mortality/survival model
   - Logistic survival equation
   - Coefficients: VARADJ (shade tolerance), MORT_B0-2
   - Two public subroutines: background_mortality, compute_survival

4. template_htdbh.f90 (295 lines)
   - Height-diameter relationships
   - Curtis-Arney equation: H = 4.5 + exp(b0 + b1/DBH^b2)
   - Coefficients: b0, b1, b2 (3 per species)
   - Subroutines: height_from_diameter, diameter_from_height (inverse)

5. template_blkdat.f90 (240 lines)
   - Species coefficient initialization (BLOCK DATA equivalent)
   - Arrays: BKRAT (bark ratios), COR2, HCOR2, RCOR2 (bias corrections)
   - Initialization and validation routines
   - Summary printing subroutine for QA/QC

6. template_sitset.f90 (365 lines)
   - Site index and stand density setup
   - Arrays: SDICON (max SDI by species), default SI values
   - Multiple subroutines:
     * initialize_site_index_defaults
     * site_index_conversion (species-to-species)
     * si_from_site_variables (estimate from elevation, aspect, soil)
     * sdindex_from_si

7. template_PRGPRM.F90 (140 lines)
   - Program parameters module
   - MAXSP, MAXTRE, MAXPLT, MAXCYC, etc.
   - Complete documentation of each parameter
   - Guidance on customization for different regions

KEY MODELS IMPLEMENTED
======================

1. Crown Ratio Model
   - Weibull-type function
   - Species and competition sensitive
   - 4 coefficients per species

2. Diameter Growth Model
   - Log-linear mixed-effects form
   - Site index and BAL effects
   - Annual iteration approach

3. Mortality/Survival Model
   - Logistic (binomial) response
   - Shade tolerance species-specific effect
   - Height-based competition

4. Height-Diameter Model
   - Curtis-Arney asymptotic form
   - Species-specific
   - Bidirectional (H from D or D from H)

5. Site Index Management
   - Default productivity values
   - SDI maximum values (self-thinning)
   - Optional SI conversion tables
