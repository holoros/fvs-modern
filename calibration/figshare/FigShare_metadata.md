## Description
This repository accompanies the manuscript "Forest relative density predicts the probability, type, and severity of disturbance across the conterminous United States" by Weiskittel et al., submitted to PNAS. It contains derived data tables, figure and table reproduction scripts, prerendered figures, and the complete HPC analysis pipeline documentation needed to reproduce all results in the main text and SI Appendix.

Contents:
- 17 CSV data files (derived outputs from the Bayesian hierarchical analysis)
- reproduce_figures.R (reproduces all main text and SI figures)
- reproduce_tables.R (reproduces all manuscript and SI tables)
- 16 HPC analysis pipeline scripts (for reference/transparency)
- 15 prerendered figure files (PNG)
- README.md and DATA_DICTIONARY.md

Raw FIA data are publicly available from the USDA Forest Service DataMart. TreeMap raster products are available from Chivhenge et al. (2025, Scientific Data 12:1728).
